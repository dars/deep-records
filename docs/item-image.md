# Item Image Spec

所有 Item Image 必須遵守以下規範。

## 尺寸

1024 x 1024

## 比例

1:1

## 格式

WebP

## 背景

固定使用 app 的墨綠背景。item-bg.png

不可透明。

不可白色背景。

不可棋盤背景。

## 構圖

物件置中。

佔畫面 60~70%。

保留呼吸空間。

## 光線

柔和。

無聚光燈。

無戲劇光。

像放在展示桌上的證物。

## 配色

墨綠
灰黑
黃銅
木頭
舊紙

避免鮮豔顏色。

## 風格

寫實

電影道具

台灣生活感

微克蘇魯

不要卡通。

不要 Fantasy。

不要遊戲 ICON。

## 禁止

不要白底

不要透明棋盤

不要陰影

不要漂浮

不要景深

不要 Logo

不要文字
